Project Overview

ReStore is an ongoing e-commerce platform project that I am building to strengthen my skills in full-stack web development. This project showcases my ability to design and implement a functional, user-friendly online shopping website.

Currently, the project includes core features such as browsing products, adding items to the shopping cart, and viewing product details. While the basic structure is in place, there are additional functionalities I plan to implement in the future to make it more comprehensive.

Key Features

Frontend

  • Built with React & TypeScript: Ensures type safety and maintainable code.
  • Responsive Design with MUI: Provides a visually appealing interface across desktop and mobile devices.
  • State Management with Redux: Efficiently handles the global state across the application.
  • Theme Switching: Offers both dark and light modes for a customizable user experience.

Backend

  • C# and .NET 6: Provides a reliable backend framework for handling data and APIs.
  • EF Core with SQLite: Ensures efficient data storage and CRUD operations.
  • Real-Time Communication with WebSockets: Adds dynamic features for seamless updates.

Planned Features

While the current version of ReStore includes essential e-commerce functionalities, there are additional features I plan to incorporate:

  • User Authentication: Implementing a registration and login system for personalized user experiences. However, since I have already built a user authentication system in another project, I might reuse parts of that implementation instead of starting from scratch.
  • Order History and Tracking: Allowing users to view past orders and track current purchases.
  • Payment Integration: Adding secure payment gateways for seamless transactions.

These enhancements will further showcase my ability to create robust, full-featured web applications.
